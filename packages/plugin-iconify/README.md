# `@vuepress-plume/vuepress-plugin-iconify`

## Install
```
yarn add @vuepress-plume/vuepress-plugin-iconify
```
## Usage
``` js
// .vuepress/config.js
const iconifyPlugin = require('@vuepress-plume/vuepress-plugin-iconify')
module.exports = {
  //...
  plugins: [
    iconifyPlugin()
  ]
  // ...
}
```
