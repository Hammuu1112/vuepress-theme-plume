# `@vuepress-plume/vuepress-plugin-notes-data`

## Install
```
yarn add @vuepress-plume/vuepress-plugin-notes-data
```
## Usage
``` js
// .vuepress/config.js
const notesDataPlugin = require('@vuepress-plume/vuepress-plugin-notes-data')
module.exports = {
  //...
  plugins: [
    notesDataPlugin()
  ]
  // ...
}
```
